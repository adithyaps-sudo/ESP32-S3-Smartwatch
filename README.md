# ESP32-S3-Smartwatch
Developed an ESP32-S3 based Smartwatch Prototype focused on health monitoring and activity tracking using embedded systems and IoT technologies.

## Duration
10 Days

## Implemented Features
- Heart Rate and SpO2 Monitoring using MAX30102 Sensor
- Accelerometer-based Step Detection using MPU6500
- Multi-screen Smartwatch User Interface using ST7789 Display
- Physical Button-based Menu Navigation System
- Real-time Motion and Activity Tracking

## System Architecture & Future Enhancements
- Designed BLE communication architecture using NimBLE
- Structured WiFi and Flask backend integration framework
- Planned AI-based Health Summary and Smart Insights System

## Technical Stack
ESP32-S3, Embedded C++, Arduino Framework, MAX30102, MPU6500, ST7789, BLE, WiFi, Flask, JSON, IoT, Embedded Systems

## Project Status
The smartwatch UI, heart rate monitoring, SpO2 sensing, and accelerometer-based activity tracking are functional. BLE connectivity, backend integration, and AI-powered health analysis are currently under development for future versions.

## Learning Outcome
This project provided practical experience in embedded systems development, sensor integration, smartwatch UI design, IoT communication, and wearable system implementation.
